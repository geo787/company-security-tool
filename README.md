# Network Security Scanner

A Python-based network port scanner designed for security testing and diagnostics.

## Features

- DNS resolution
- Multithreaded port scanning
- CLI interface
- CSV report generation
- Fast scanning performance

## Usage

python scanner.py google.com

python scanner.py google.com --start 20 --end 100

## Output

Reports are saved in the reports folder.
